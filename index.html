Structure de base

  • Header

    Représente l'en-tête du site.
    Contient, par exemple : le titre principal, la bannière, un bouton, ...

  • Navigation

    Contient le menu de navigation.
    Le menu est généralement constitué d'une liste de lien.
    Peut-être placé sur le coté, avant ou après le header.

    Un document HTML peut contenir plusieurs élements nav.

  • Main

    Premier élement principal contenant le corps du site.
    On y retrouve les différentes sections, articles, ...
    Personnellement, je l'utilise énormement comme conteneur pour flexbox ou grid.

    Un document HTML ne peut contenir qu'un seul élement main.

  • Section

    Second élement principal comprennant les textes, images, ... Peut être composé de plusieurs articles ou paragraphes.

    Un document HTML peut être composé de plusieurs sections.

  • Article

    Element contenant, comme son l'indique, un article. Il offre à l'utilisateur un ensemble d'informations sous forme textuelle.
    Peut être composé de plusieurs paragraphes ou citations.

  • Aside

    Element secondaire offrant un complément d'information.
    On peut y retrouver par exemple un fil d'actualité, les promotions, la biographie de l'auteur, ...

  • Footer

    Situé à la fin de la page, il fournit des informations complémentaires.
    Généralement, on y retrouve les informations de contact, les obligations légales, ...
